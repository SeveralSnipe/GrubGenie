CommerceNearMeRoutes

CommerceNearMeRouter

GET "CommerceNearMeRouter/AllStores"

Retrieves all store details in alphabetical order.

POST "CommerceNearMeRouter/foodNearMeDist"

Request Body:

{
    location: [latitude, longitude],
    maxDist: number, // Maximum distance from user
    sort: "dist" or "name" // Sorting result based on name or distance from user
}

Response: Store details

POST "CommerceNearMeRouter/foodNearMeItem"

Request Body:

{
    location: [latitude, longitude],
    maxDist: number, // Maximum distance from user :- default dist 5kms
    sorts: "dist" or "name" or "price",  :- default dist
    Item: "Chicken"
}

Response: Retrieves stores with specified items.

CommerceRequestRouter

Request Body:

{
    userId: string,
    Orders: {
        storeId: [itemids, ...]

}

Makes a request entry in the request table.
